About me

Hi, I'm Shin Yin. A discoverer and an adventurer from Melaka, a historical state in Malaysia rich with heritage buildings, ancient landmarks, and colonial structures. I came to Minnesota in 2017 to pursue my education and to experience the "Minnesota Nice" and its cold weather.

I graduated from the University of Minnesota-Twin Cities with a Bachelor's degree in Computer Science in Fall 2019. During the school years, I have taken Program Design & Development, Practice of Database System, Advanced Algorithms & Data Structures, Operating System, Advanced Programming Principles, Artificial Intelligence, Internet Programming, Machine Architecture and Organization, etc.

I am passionate about Web/Application/Software Development and actively looking for Software Developer/Engineer jobs to showcase my skills and to continue to enrich my knowledge.

This software project is to develop a transit simulator in which simulations can be controlled via external configuration (without code change) and be visualized within a graphics window.   • Personal Schedule Website

    Personal Schedule Website is a course (CSci 4131) project at the University of Minnesota - Twin Cities that not only allow users to add or change their events but also to locate and navigate to events location using Geo-location APIs. It also has additional functionality where users can get stock data from selected companies using Alpha Vantage APIs. This project is implemented in HTML & CSS, JavaScript, jQuery, JSON/AJAX, XML, and the database MySQL. It has 3 server-side developments: Python, Node.js, and PHP.
